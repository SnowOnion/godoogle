Godoogle

Godoogle is a Go API search engine, which allows you to search by approximate function type signature, including generic functions.

Examples

  • func(string) (int, error)
  • func(string) int
  • [a comparable] func(...a) a
  • func ([]string, string) string
  • func(strs []string, sep string) string
  • [T any] func(bool, T, T) T
  • [T any] func(bool, func() T, func() T) T
  • [a,b any] func([]a, func(a, int) b) []b
  • [T any] func (f func() T) <-chan T
